5.Iteration Method

1. Rewrite the Equation:
Start with an equation f(x) = 0 and rewrite it in the form x = g(x). This form is crucial for the iterative process.

2. Choose an Initial Guess:
Select an initial value, x0, which is an educated guess close to the expected root.

3. Iterative Calculation:
Apply the function g(x) to the current value, xn, to obtain the next approximation, xn+1: xn+1 = g(xn).

4. Repeat the Process:
Continue this iterative process, using the newly calculated value (xn+1) as the input for the next iteration, until the difference between consecutive approximations is smaller than a specified tolerance.

5. Convergence:
The method aims to converge towards the fixed point, meaning the sequence of values {xn} will approach a value that satisfies the equation x = g(x).

6. Convergence Condition:
he method typically converges if |g'(x)| < 1 for the value of x around the fixed point.
