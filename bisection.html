1.Bisection Method

1.Initial Interval: Choose two points, a and b, such that f(a) and f(b) have opposite signs, indicating that a root lies between them.

2.Midpoint Calculation: Calculate the midpoint c of the interval: c = (a + b) / 2.

3.Function Evaluation: Evaluate the function f(c) at the midpoint.

4.Interval Refinement:
If f(c) = 0, then c is the root.
If f(c) has the same sign as f(a), then the root lies in the interval [c, b], so update a to c.
If f(c) has the same sign as f(b), then the root lies in the interval [a, c], so update b to c.

5.Iteration: Repeat steps 2-4 until the interval size becomes small enough or the desired accuracy is achieved.
